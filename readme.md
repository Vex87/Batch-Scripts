# Batch Scripts

## COMMAND PROMPT (CMD)
##### echo CD ..                          
Sets directory to parent.
##### echo CD \                           
Sets directory to CL.
##### echo CD `LOCATION`                  
Sets directory.
##### echo CLS                            
Clear CL.
##### echo CMD                            
Starts a new instance of CL.
##### echo DIR                            
Return files of directory.
##### echo ECHO `MSG`                     
Returns message.
##### echo EXIT                           
Closes CL.
##### echo NEWFILE                        
Opens CL.
##### echo RUNADMIN                       
Starts a new instance of CL as an admin.

## PC
##### echo ENDTASK                        
Terminates a task.
##### echo IPCONFIG                       
Returns IP address.
##### echo LOGOFF                         
Logs the current user out of the machine.
##### echo PING `URL`                     
Allows the user to test the availability of a network connection to a specified host.
##### echo SYSTEMINFO                     
Displays system info.
##### echo TASKLIST                       
Displays all currently running tasks including services.

## OTHER
##### echo HIS                            
View command history.
##### echo SHUTDOWN                       
Allows proper local or remote shutdown of machine.
##### echo SORT                           
Insensitive case filtering.
##### echo SYNC                           
Sync local repo to system32 (Security: Admin).
##### echo TREE                           
Graphically displays the directory structure of apath.
##### echo TYPE                           
Displays the contents of a text file.

## FOLDERS (FOL)
##### echo DELFOL                         
Deletes folder.
##### echo MD `NAME`                      
Creates folder.
##### echo OPEN                           
Opens a folder.

## FILES
##### echo COMP                           
Show differences between any files.
##### echo DEL `NAME`                     
Deletes file.
##### echo FC                             
Show differences between any files.
##### echo NEWFILE                        
Creates file.
##### echo NEWTXTFILE `NAME`              
Creates file with text.
##### echo XCOPY `FILE` `DESTINATION`     
Copy all descendants of path.

## FOLDERS AND FILES (FAF)
##### echo COPY `FILE` `DESTINATION`      
Copy file.
##### echo MOVE `FILE` `DESTINATION`      
Move file.
##### echo REN `FILE` `NAME`              
Renames file.
##### echo START `FILE`                   
Opens/runs folder/file.

## GOOGLE
##### echo EXITCHROME                     
Closes chromne.
##### echo START CHROME `URL`             
Search chrome.
##### echo START CHROME --INCOGNITO       
Starts chrome in incognito mode.

## VISUAL STUDIO CODE (VSC)
##### echo CODE -D `FILE` `FILE`

##### echo CODE -H                        
Prints commands.
##### echo CODE -NAME                     
Opens new session.
##### echo CODE .                         
Open VSC.
##### echo CODE `FILE`                    
Opens new file.
